## Overview

This project centralizes documentation and research for the Call of Duty franchise, specifically covering entries from Call of Duty 4: Modern Warfare to Call of Duty 10. It consolidates technical information about game internals, modding approaches, and reverse engineering discoveries that were previously scattered across forums, personal archives, and defunct websites.

The repository preserves critical technical knowledge that might otherwise disappear as communities evolve and older resources become unavailable. Whether you are a modder seeking to understand engine behavior, a researcher analyzing game architecture, or a curious player exploring how these games work, this wiki offers structured, peer-reviewed documentation to support your efforts.

## Key Features

- **Game-Specific Technical References** – In-depth documentation for CoD 4 through CoD 10 titles
- **Reverse Engineering Records** – Organized findings from binary analysis and memory research
- **Community-Verified Information** – Knowledge contributed and validated by active modding groups
- **Searchable Wiki Structure** – Browseable through categorized topics with cross-referenced articles
- **Legacy Content Preservation** – Archived documentation from now-inaccessible resources and forums
- **Open Contribution Model** – Anyone can submit corrections, additions, or new research results

## Installation

Clone the repository to your local system:

```bash
git clone https://github.com/brandonhub1993/cod-research-hub.git
cd CoD-Research
```

The wiki consists of static HTML files. Open `index.html` in any modern web browser to begin browsing. No server software or build tools are required for basic viewing.
